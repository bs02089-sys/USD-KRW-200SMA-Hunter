# SmartFXAllocator

환율 변동성 기반 **정기 적립식 + 추가 매수 전략 자동화 시스템**  
매월 셋째 주 목요일 정기 매수와, 최근 6개월 변동성을 반영한 추가 매수 기준을 자동으로 계산합니다.  
투자 규율을 코드로 구현하여 장기적으로 안정적인 달러 매수를 지원합니다.

---

## 🚀 주요 기능
- 📅 **정기 적립식 매수일 계산**  
  - 매월 셋째 주 목요일, 330,000원 자동 매수
- 📊 **변동성 기반 추가 매수**  
  - 최근 6개월 환율 데이터를 기반으로 1~3차 기준점 자동 계산
  - 기준점 이하일 경우, 단계별로 167,000원씩 추가 매수
- 🔔 **디스코드 알림 연동**  
  - 매수일 및 기준 충족 여부를 실시간 알림으로 확인 가능
- 🌐 **깃허브 자동 푸시**  
  - 매수 내역을 로그 파일에 기록하고 자동 커밋·푸시
